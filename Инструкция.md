# Основные команды для работы с Git

## Создание Git-репозитория

1. Команда *git init* - указываем папку, в которой 
git начнёт отслеживать изменения.

2. Команда *git status* - Показывает текущее состояние гита, есть ли изменения, которые нужно сохранить.

3. Команда *git commit* - сохраняет данные 
во внутренней базе данных, а затем сдвигает указатель текущей ветки на этот слепок.

4. Команда *git log* - Журнал изменений.

5. Команда *git checkout* - Переключение между версиями. 

6. *git log* - Команда,отображающая логи коммитов.

7. *git add file_name* - команда,добовляющая актуальную версию указанного файла

## Базовые команды со второго семинара

* *git branch* - команда,отображающая сущетсвующие ветки репозитории

* *git branch branch_name# - команда,создающая новую ветку в репозитории

* *git checkout branch_name* - команда,переключения на указанную ветку

* *git log --graph* - команда,для отображения коммита в виде дерева

## Базовые команды с третьего семинараы

* *git spoiler one* - первая супер полезная команда

* *git spoiler two* - вторая супер полезная команда
